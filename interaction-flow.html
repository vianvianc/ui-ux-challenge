Interaction Flow

Ella visits wunderground.com homepage

  • Ella clicks Search Locations
  • Dialog box opens for user to type in Search Location
  • Ella Types: Rifle
  • Dialog box auto suggests Rifle,CO
  • Ella scrolls to select Rifle,CO
  • Page for Rifle,CO loads
  • Ella clicks the hourly tab
  • Tab displays hourly information for selected day
  • Ella clicks Next days hourly forecast to see the next day
  • Next day hourly forecast loads
  • Ella clicks the Star next to Rifle, CO Hourly Forecast
  • Star turns golden from clear
  • Location is saved in cache
